# RFID_App_IoT
A small IoT demo where a Raspberry pi 3b+ and an Android Application share RFID data through Google Firestore database

Equipment: <br/>
-Raspberry Pi 3B+ <br/>
-RFID-RC522 Reader <br/>
-Google Firestore Database <br/>
-Android Studio
